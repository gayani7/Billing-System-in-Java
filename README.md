# Billing System – City Mart

A Java-based Billing System developed using Apache NetBeans IDE.
This project is designed for small retail shops to manage product billing and payment calculations..

## Features

- Add product name, price, and quantity
- Automatically calculate total amount
- Add multiple products to a bill
- Calculate subtotal, paid amount, and remaining balance
- Print the final bill
- Simple and user-friendly GUI (Java Swing) connectivity.

## Technologies Used
- Java
- Java Swing (GUI)
- Apache NetBeans IDE
